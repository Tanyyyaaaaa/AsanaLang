# 🎓 Путь ученика (Tutorial)

Добро пожаловать в ваше первое руководство по ASANALANG. Мы пройдем путь от установки инструментов до создания сложной логики.

## Шаг 1: Подготовка окружения
Поскольку интерпретатор написан на F#, вам понадобится **.NET SDK**.
1. Склонируйте репозиторий: `git clone [url]`
2. Перейдите в папку: `cd AsanaLang`
3. Восстановите зависимости: `dotnet restore`

## Шаг 2: Первая мантра (Hello World)
Создайте файл `hello.asana` и впишите туда:
```lisp
(om main ()
  (exhale (mantra "Намасте, мир!")))
```
Запустите программу командой:
`dotnet run -- hello.asana`

## Шаг 3: Объявление сущностей (Nirvana)
В AsanaLang мы не просто создаем переменные, мы приводим их к состоянию покоя через `nirvana`:
```lisp
(nirvana age 25)
(exhale age)
```

## Шаг 4: Сложные формы (Функции)
Для создания логики используйте `asana` (именованная функция) или `mudra` (анонимная).
```lisp
(asana square (x) (* x x))
(exhale (square 5)) ;; Выведет 25
```

## Шаг 5: Просветление (Рекурсия)
Чтобы решить сложную задачу, нужно пройти через цикл `samsara`. Посмотрите на пример факториала в разделе `examples.md`, чтобы увидеть, как функции перерождаются до достижения результата.
