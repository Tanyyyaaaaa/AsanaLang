# 🕉 Справочник Практик

| Тип AsanaLang | Тип в .NET | Описание | Лимит / Формат |
| :--- | :--- | :--- | :--- |
| **mountain** | `Int32` | Целые знаковые числа | от -2^31 до 2^31-1 |
| **water** | `Double` | Числа с плавающей точкой | 64-bit IEEE 754 |
| **mantra** | `String` | Строковые литералы | UTF-16 в кавычках `""` |
| **balance** | `Boolean` | Логические значения | `true` или `false` |
| **forest** | `List<Expr>` | Связный список | Рекурсивная структура |


### Конструкции
* **(nirvana x 10)** — Привязка значения к имени.
* **(if_calm cond then else)** — Ветвление в состоянии спокойствия.
* **(dhyana expr1 expr2 ...)** — Последовательное выполнение команд (медитация).
* **(namaste expr)** — Завершение практики и возврат результата.

### Рекурсия (Samsara)
В AsanaLang рекурсия реализуется через цикл перерождений `samsara`. Это мощный инструмент, позволяющий функциям вызывать самих себя.
