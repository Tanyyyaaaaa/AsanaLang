# 🕉 Справочник Практик

### Типы Данных
* `mountain` — Целые числа (опора).
* `water` — Числа с плавающей точкой (текучесть).
* `mantra` — Строки (звуковые вибрации).
* `balance` — Логический тип `true/false`.

### Конструкции
* **(nirvana x 10)** — Привязка значения к имени.
* **(if_calm cond then else)** — Ветвление в состоянии спокойствия.
* **(dhyana expr1 expr2 ...)** — Последовательное выполнение команд (медитация).
* **(namaste expr)** — Завершение практики и возврат результата.

### Рекурсия (Samsara)
В AsanaLang рекурсия реализуется через цикл перерождений `samsara`. Это мощный инструмент, позволяющий функциям вызывать самих себя.
