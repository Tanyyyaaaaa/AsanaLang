# 🧘 Философия Языка

Язык программирования не должен быть источником стресса. ASANALANG переосмысляет стандартные концепции:


| Стандарт | В AsanaLang | Философия |
| :--- | :--- | :--- |
| `let / var` | `nirvana` | Переменная достигает покоя (состояния). |
| `function` | `asana` | Устойчивая поза логики. |
| `lambda` | `mudra` | Эфемерное движение данных. |
| `recursion` | `samsara` | Перерождение кода до момента просветления. |
