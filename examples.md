# 📜 Примеры Практик

### 1. Числа Фибоначчи
Этот пример показывает использование вложенных `mudra` и рекурсии `samsara`.

```lisp
(om main ()
  (dhyana
    (nirvana fib
      (samsara
        (mudra (f)
          (mudra (n)
            (if_calm (< n 2)
              n
              (+ (f (- n 1)) (f (- n 2)))
            )
          )
        )
      )
    )
    (exhale (fib 7))
  )
)
```

### 2. Священные рощи (Списки)
Списки в AsanaLang обрабатываются набором встроенных практик:
* `(head list)` — Первый элемент.
* `(mirror list)` — Реверс списка.
* `(push 5 list)` — Добавление нового ростка (элемента).
